Scratch Project:
Duration: Approximately 7 days

Objective: To create an interactive game or story using the features of Scratch

Course: Computer Science Principles

Project: The main idea of this project involved creating a fully functioning game or story with user interaction. Different tools and techniques used in Scratch were researched and learned about. In this specific game, the user follows along a basic storyline revolving around a high schooler. The user is able to control the choices that their character makes by selecting the scenario they want to put the character into. To the right of this text, is a demo of the storyline. To begin click the green flag, and the character will walk you through the story that you will be able to control.
